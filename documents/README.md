# Hand sized Universe

手のジェスチャーで操作できる3D太陽系シミュレーションアプリケーション

![Version](https://img.shields.io/badge/version-1.1-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

## 概要

**Hand sized Universe**は、Webカメラを使用した手の動き検出により、直感的に太陽系を探索できる3Dシミュレーションアプリケーションです。MediaPipeとThree.jsを使用して、リアルタイムで手のジェスチャーを認識し、3D空間内の太陽系を操作できます。

## 主な機能

### 🎮 手のジェスチャー操作

- **左手**: カメラ回転とズーム（ピンチジェスチャー）
- **右手**: 手をかざして星の説明を表示
- **リアルタイム検出**: MediaPipe HandLandmarkerによる高精度な手の検出

### 🌍 2つのカメラモード

- **太陽系全体モード**: 太陽系全体を俯瞰できる視点
- **地球視点モード**: 地球の表面から外側を見る視点

### ✨ 星の説明表示

- 右手を惑星の上にかざすと、その惑星の詳細な説明が表示されます
- 0.5秒間同じ惑星に手をかざし続けると説明が表示されます
- 説明パネルは30秒後に自動的に消えます
- 選択された惑星はハイライト表示されます

### 🪐 リアルタイムシミュレーション

- 太陽、9つの惑星、月、ISSのリアルタイムアニメーション
- 公転・自転の現実の比率に基づいたシミュレーション
- 6,000個の星が瞬く背景

## デモ

### Webアプリとして実行

ローカルサーバーを起動して、ブラウザで`index.html`を開いてください。

```bash
# Python 3の場合
python -m http.server 8000

# Node.jsの場合
npx http-server
```

その後、ブラウザで `http://localhost:8000` にアクセスしてください。

### Androidアプリとして実行

このアプリはCapacitorを使用してAndroidアプリとしてもビルドできます。詳細は [ANDROID_APP_SETUP.md](ANDROID_APP_SETUP.md) を参照してください。

```bash
# 依存関係のインストール
npm install

# Androidプロジェクトの追加
npx cap add android

# アプリの同期
npx cap sync

# Android Studioで開く
npx cap open android
```

## 操作方法

### 左手の操作（太陽系全体モード）

1. **カメラ回転**: 左手を画面の前で動かすと、カメラが手の動きに追従して回転します
2. **ズーム**: 
   - 親指と人差し指でピンチ（つまむ）→ ズームイン
   - 指を広げる → ズームアウト
   - ズーム範囲: 10 ～ 1000

### 右手の操作（太陽系全体モード）

3. **星の説明表示**:
   - 右手を惑星の上にかざすと、最も近い惑星が自動的に選択されます
   - 0.5秒間同じ惑星に手をかざし続けると、説明パネルが表示されます
   - 説明パネルは30秒後に自動的に消えます
   - 手動で閉じる場合は、パネル右上の×ボタンをクリック

### UI操作

4. **カメラモード切り替え**: UIの「カメラ」トグルスイッチをクリック
5. **ラベル表示切り替え**: UIの「ラベル」トグルスイッチをクリック

## 開発

### セットアップ

1. リポジトリをクローン
2. ローカルサーバーを起動
3. ブラウザで`index.html`を開く

### 設定パラメータ

`script.js`の`SETTINGS`オブジェクトで調整可能:

```javascript
const SETTINGS = {
    cameraSmooth: 0.05,      // カメラ追従の滑らかさ
    zoomSmooth: 0.1,         // ズームの滑らかさ
    minZoom: 10,             // 最小ズーム
    maxZoom: 1000,           // 最大ズーム
    starCount: 6000,         // 星の数
    planetEmissiveIntensity: 0.1  // 惑星の自発光の強さ
};
```

### デバッグ

- ブラウザの開発者ツール（F12）でコンソールログを確認
- パフォーマンス分析ツールでフレームレートを確認

## トラブルシューティング

### よくある問題

1. **カメラが検出されない**
   - ブラウザのカメラ権限を確認
   - 他のアプリケーションがカメラを使用していないか確認

2. **パフォーマンスが低い**
   - GPUアクセラレーションが有効か確認
   - ブラウザのハードウェアアクセラレーション設定を確認

3. **テクスチャが表示されない**
   - `assets`フォルダにテクスチャ画像が存在するか確認
   - ブラウザのコンソールでエラーを確認
