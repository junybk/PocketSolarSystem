# Hand sized Universe - システム仕様書

## 概要

**Hand sized Universe**は、手のジェスチャーで操作できる3D太陽系シミュレーションアプリケーションです。Webカメラを使用した手の動き検出により、直感的に太陽系を探索できます。

**最終更新**: 2025年12月4日

## システム概要

### 目的

- 手のジェスチャーによる直感的な3D太陽系探索
- 現実の比率に基づいた正確な太陽系シミュレーション
- 教育・学習用途での太陽系の理解促進

### 特徴

- **手のジェスチャー操作**: Webカメラを使用した手の動き検出
- **2つのカメラモード**: 太陽系全体視点と地球視点
- **リアルタイムアニメーション**: 公転・自転のリアルタイムシミュレーション
- **現実の比率**: 惑星のサイズ、軌道距離、自転速度が現実の比率に基づく
- **GPUレンダリング**: WebGLによる高速3Dレンダリング

## 主要機能

### 1. 手のジェスチャー操作

システムは**左右の手を分離**して認識し、それぞれ異なる操作を割り当てています：

#### 左手の操作
- **手の動き**: カメラの回転（上下左右）
- **ピンチ（つまむ）**: ズームイン/アウト
- **リアルタイム検出**: MediaPipe HandLandmarkerによる高精度な手の検出
- **視覚的フィードバック**: 左手カーソル（シアン色）が画面上に表示される

#### 右手の操作
- **手をかざす**: 星の説明を表示
  - 右手を惑星の上にかざすと、最も近い惑星が自動的に選択される
  - 0.5秒間同じ惑星に手をかざし続けると説明が表示される
  - 選択された惑星はハイライト表示される（青みがかった色）
  - 説明パネルは30秒後に自動的に非表示になる
  - 別の惑星に手をかざすと、説明が切り替わる（フェードアウト/フェードインアニメーション付き）
- **視覚的フィードバック**: 右手カーソル（オレンジ色）が画面上に表示される

### 2. カメラモード

- **太陽系全体モード**: 太陽系全体を俯瞰できる視点
  - ジェスチャー操作でカメラを回転・ズーム可能
  - 地球視点カメラの位置と向きをマーカーで表示
  
- **地球視点モード**: 地球の表面から外側を見る視点
  - カメラ位置と向きが固定（地球の自転に追従）
  - ジェスチャー操作は無効化
  - 太陽系全体モードのジェスチャー操作の影響を受けない

### 3. 星の説明表示

- **手をかざして選択**: 右手を惑星の上にかざすことで、その惑星の説明を表示
- **自動選択**: レイキャスティングにより、手の位置から最も近い惑星を自動的に選択
- **ハイライト表示**: 選択された惑星は視覚的にハイライトされる（青みがかった色）
- **説明パネル**: 
  - 星の名前と詳細な説明文を表示
  - スライドインアニメーションで表示
  - 切り替え時はフェードアウト/フェードインアニメーション
  - 30秒後に自動的にフェードアウトで非表示
  - 手動で閉じることも可能（×ボタン）
- **対応天体**: 太陽、すべての惑星、月、ISS

### 4. ラベル表示

- すべてのオブジェクト（太陽、惑星、月、ISS）に名称ラベルを表示
- トグルスイッチで表示/非表示を切り替え可能
- CSS2DRendererによる常にカメラを向くラベル表示

### 5. 太陽系シミュレーション

- **太陽**: 中心に配置、脈動アニメーション
- **9つの惑星**: 水星、金星、地球、火星、木星、土星、天王星、海王星、冥王星
- **月**: 地球の周りを公転
- **ISS**: 地球の周りを公転する人工衛星
- **背景の星**: 6,000個の星が瞬きアニメーション

## 操作方法

### 基本操作

#### 左手の操作（太陽系全体モードでのみ有効）

1. **カメラ回転**
   - 左手を画面の前で動かすと、カメラが手の動きに追従して回転します
   - 手の位置に応じて、カメラが上下左右に回転します
   - 太陽系全体モードでのみ有効

2. **ズーム**
   - 左手で親指と人差し指でピンチ（つまむ）ジェスチャーを行うとズームイン
   - 指を広げるとズームアウト
   - 太陽系全体モードでのみ有効
   - ズーム範囲: 10 ～ 1000
   - ピンチ中は左手カーソルが緑色に変化

#### 右手の操作（太陽系全体モードでのみ有効）

3. **星の説明表示**
   - 右手を惑星の上にかざすと、最も近い惑星が自動的に選択されます
   - 0.5秒間同じ惑星に手をかざし続けると、説明パネルが表示されます
   - 選択された惑星はハイライト表示されます（青みがかった色）
   - 別の惑星に手をかざすと、説明が切り替わります
   - 説明パネルは30秒後に自動的にフェードアウトで消えます
   - 手動で閉じる場合は、パネル右上の×ボタンをクリック

#### UI操作

4. **カメラモード切り替え**
   - UIの「カメラ」トグルスイッチをクリック
   - 「太陽系全体」と「地球視点固定」を切り替え

5. **ラベル表示切り替え**
   - UIの「ラベル」トグルスイッチをクリック
   - すべてのオブジェクトのラベルを表示/非表示

### 操作の制限

- **地球視点モード**: ジェスチャー操作（左手・右手）は無効化されます
- **カメラの初期位置**: 太陽系全体モードに戻ると、カメラ位置がリセットされます
- **説明表示**: 太陽系全体モードでのみ有効（地球視点モードでは無効）

## カメラシステム

### アーキテクチャ

システムは**2台の独立したカメラ**を使用しています：

1. **太陽系全体カメラ (`solarCamera`)**
   - `orbitGroup`の子要素として配置
   - ジェスチャー操作で回転・ズーム可能
   - 初期位置: (0, 0, 60)

2. **地球視点カメラ (`earthCamera`)**
   - `scene`に直接追加（`orbitGroup`の影響を受けない）
   - 地球の表面に固定、地球の自転に追従
   - 太陽系全体モードのジェスチャー操作の影響を受けない

### カメラ切り替え

- `activeCamera`変数で現在使用中のカメラを管理
- モード切り替え時に自動的に適切なカメラに切り替わる

### 地球視点カメラの位置計算

- 地球のローカルX軸方向（地球の自転に追従）
- 地球の表面から半径+0.01の位置に配置
- `calculateEarthCameraPositionAndDirection()`関数で計算
- マーカーと矢印で位置と向きを可視化

## 表示オブジェクト

### 太陽系オブジェクト

詳細は`SOLAR_SYSTEM_SPECS.md`を参照してください。

#### 主要オブジェクト

- **太陽**: 半径5.0、原点に配置、脈動アニメーション
- **9つの惑星**: 現実の比率に基づくサイズと軌道距離
- **月**: 地球の周りを公転、潮汐ロック
- **ISS**: 地球の周りを公転

### 視覚要素

- **軌道線**: 各惑星の軌道をリングで表示
  - ズームに応じて太さが動的に調整
  - 透明度: 0.3

- **マーカー**: 地球視点カメラの位置を示す
  - シアンの球体とリング
  - 太陽系全体モードでのみ表示

- **矢印**: 地球視点カメラの向きを示す
  - シアンの矢印
  - 両方のモードで表示（地球視点モードではカメラの向きと一致）

- **ラベル**: オブジェクト名を表示
  - CSS2DObjectによる常にカメラを向く表示
  - フォントサイズ: 0.875rem

- **手のカーソル**: 手の位置を示すマーカー
  - 左手カーソル: シアン色（カメラ回転・ズーム操作中）
  - 右手カーソル: オレンジ色（星の説明選択中）
  - ピンチ中は左手カーソルが緑色に変化

- **ハイライト**: 選択された惑星の視覚的フィードバック
  - 青みがかった色でハイライト表示
  - 手をかざしている間、常に更新される

- **説明パネル**: 星の詳細情報を表示
  - スライドインアニメーションで表示
  - 切り替え時はフェードアウト/フェードインアニメーション（スケール変化付き）
  - 30秒後に自動的にフェードアウトで非表示
  - 閉じるボタン（×）で手動で閉じることも可能

### 背景

- **星**: 6,000個の星が3D空間にランダム配置
- **フォグ**: 指数フォグで深度感を演出
- **瞬きアニメーション**: GPUシェーダーによる星の瞬き

## アニメーション

### 公転

- 各惑星が太陽の周りを公転
- 公転速度は惑星ごとに異なる（現実の比率）
- `pivot.rotation.y`で制御

### 自転

- すべての惑星と月が自転
- 自転速度は現実の比率に基づく（地球を1.0とした相対値）
- `mesh.rotation.y`で制御
- 金星は逆回転（retrograde）

### 特殊なアニメーション

- **太陽**: 脈動アニメーション（スケール変化）
- **星**: 瞬きアニメーション（シェーダー）
- **軌道線**: ズームに応じた太さの動的調整

### アニメーションループ

- `requestAnimationFrame`による60fpsのアニメーションループ
- 各フレームで以下を更新:
  1. MediaPipeによる手の検出（左右の手を分離）
  2. 左手の処理: カメラ回転とズーム
  3. 右手の処理: 手をかざして惑星を選択、説明表示
  4. カメラ位置の更新
  5. 軌道線の太さ調整
  6. 惑星の公転・自転
  7. 太陽の脈動
  8. 星の瞬き
  9. レンダリング

## 開発環境

### 必要なツール

- **テキストエディタ**: VS Code推奨

  - Python: `python -m http.server 8000`

### 設定パラメータ

`script.js`の`SETTINGS`オブジェクトで調整可能:

```javascript
const SETTINGS = {
    cameraSmooth: 0.05,      // カメラ追従の滑らかさ

    maxZoom: 1000,           // 最大ズーム
    starCount: 6000,         // 星の数
    planetEmissiveIntensity: 0.1  // 惑星の自発光の強さ
```

## 技術的な詳細

### 座標系

- **ワールド座標系**: 太陽が原点(0, 0, 0)
- **ローカル座標系**: 各惑星のローカル座標（自転軸の傾きを考慮）

### カメラの座標変換

- **太陽系全体カメラ**: `orbitGroup`のローカル座標系
- **地球視点カメラ**: ワールド座標系（`orbitGroup`の影響を受けない）

### 手の検出とジェスチャー認識

- **MediaPipe HandLandmarker**: 2つの手を同時に検出（`numHands: 2`）
- **左右の手の分離**: `handedness`情報を使用して左右を判定
- **レイキャスティング**: 右手の位置から最も近い惑星を選択
- **ハイライト処理**: 選択された惑星のマテリアルを動的に変更
- **タイマー管理**: 説明パネルの自動非表示を管理

### パフォーマンス最適化

- **インスタンシング**: 星のレンダリングに使用
- **重複フレーム処理の防止**: `video.currentTime`を使用してフレームの重複を検出
- **軌道線の動的調整**: 変化が大きい場合のみジオメトリを再作成
- **フラスタムカリング**: Three.jsが自動的に処理

## トラブルシューティング

### よくある問題

1. **カメラが検出されない**
   - ブラウザのカメラ権限を確認
   - 他のアプリケーションがカメラを使用していないか確認

2. **パフォーマンスが低い**
   - GPUアクセラレーションが有効か確認
   - ブラウザのハードウェアアクセラレーション設定を確認

3. **テクスチャが表示されない**
   - `assets`フォルダにテクスチャ画像が存在するか確認
   - ブラウザのコンソールでエラーを確認
