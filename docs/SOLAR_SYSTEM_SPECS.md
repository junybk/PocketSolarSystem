# 太陽系シミュレーション - オブジェクト仕様書

このドキュメントは、Pocket Solar System に存在する主要オブジェクトのサイズ・距離・速度の実装値をまとめたものです。

## 惑星データ（実装値）

- 水星: `r=0.38`, `dist=7.8`, `speed=0.04`, `rotationSpeed=0.017`
- 金星: `r=0.95`, `dist=14.4`, `speed=0.03`, `rotationSpeed=0.004`（逆回転）
- 地球: `r=1.0`, `dist=20`, `speed=0.025`, `rotationSpeed=1.0`
- 火星: `r=0.53`, `dist=30.4`, `speed=0.02`, `rotationSpeed=0.97`
- 木星: `r=11.2`, `dist=104`, `speed=0.01`, `rotationSpeed=2.44`
- 土星: `r=9.4`, `dist=191`, `speed=0.008`, `rotationSpeed=2.27`
- 天王星: `r=4.0`, `dist=384`, `speed=0.006`, `rotationSpeed=1.39`
- 海王星: `r=3.9`, `dist=601`, `speed=0.005`, `rotationSpeed=1.49`
- 冥王星: `r=0.19`, `dist=790`, `speed=0.004`, `rotationSpeed=0.16`

## 土星リング（実装値）

- 内側半径: `r * 1.15`（土星で `10.81`）
- 外側半径: `r * 2.4`（土星で `22.56`）

## 地球周辺オブジェクト

- 月:
  - 半径: `earth.r * 0.27`
  - 地球からの距離: `earth.r + 2.5`
  - 公転速度: `0.015`
- ISS:
  - 地球からの距離: `earth.r + 0.8`
  - 公転速度: `0.02`

## 背景

- 星の数: `SETTINGS.starCount`（現行デフォルト `10000`）
- 星はシェーダーで瞬きアニメーション

## 表現方針

- 惑星の相対サイズ・軌道距離・速度差の理解を重視
- 可視化しやすさのため、時間スケールやカメラ挙動は調整
