# Pocket Solar System - システム仕様書

## 概要

`Pocket Solar System` は、マウス/タッチ操作で太陽系を探索する 3D シミュレーションです。  
現行実装は「Web カメラ手認識」ではなく、ポインター操作を中心とした UI になっています。

## 主要機能

1. **太陽系表示**
   - 太陽、8惑星 + 冥王星、月、ISS を表示
   - 公転・自転アニメーションを継続更新

2. **カメラ操作**
   - 太陽系全体モード: ドラッグ回転 / ホイール・ピンチズーム
   - 地球視点モード: 地球表面基準の固定視点
   - サイドメニューの天体選択で対象天体へフォーカス

3. **天体情報 UI**
   - クリック/タップで説明パネル表示
   - 数値ファクト + 説明文を表示
   - 閉じるボタン対応 / 自動非表示タイマーあり

4. **ライティング**
   - 太陽位置の PointLight により昼夜面を表現
   - 惑星は太陽との位置関係に応じて明暗が動的に変化

5. **レスポンシブ UI**
   - デスクトップ: 左サイドメニュー常時表示
   - モバイル: ハンバーガーメニューで内容を開閉

## 技術構成

- Three.js (`three@0.160.0`)
- CSS2DRenderer（ラベル描画）
- JavaScript ES Modules

## 主要ファイル

- `index.html`: 画面構造
- `style.css`: サイドメニュー/パネル/レスポンシブ
- `script.js`: シーン構築、入力処理、カメラ制御、情報表示
